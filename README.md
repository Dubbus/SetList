# SetList

A platform for cataloging and sharing live Carnatic music performances.

## Setup

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:5173. Create an account to get started.

## Features

- **Add concerts** — paste a YouTube/SoundCloud link, add metadata
- **Segment** — break concerts into alapana, song, RTP, swaram, thani sections with timestamps and raga labels
- **Rate** — 1–5 star ratings per segment
- **Browse** — search and filter all concerts
- **Profile** — track your ratings and favorite artists

All data is stored in your browser's localStorage.
